Questions

1. What does the server in the client server architecture do?

A server in the client server architecture provides services or resources
to the client. The server gets a request and looks if it has the information,
if not it sends the request to another server. The server communicates with
the client over a network or internet connection.

2.What does the client do?

The client request information or services from the server or servers, a client
can be connected to serval servers at the same time.

3. What is an IP address?

An Internet Protocol (IP) address is a unique address assigned to a unit, like a
computer or printer in a TCP/IP-based network. The IP addresses are numeric
and logical, they are divided into 2 parts, the network part and the host part.
The network part shows which network the device is a part of and the host
part pinpoints the devices exact location.

6. Explain what HTML is

HTML stands for Hyper Text Markup Language and what is primarily do is to
describe the structure of a page in a text file that any browser can read.
The HTML gives you the opportunity to format text, add pictures, input forms and
tables etc. The structure of the HTML is build up on elements and tags.
